# 参考资料 · 运维 / SRE / DataOps / FinOps

## SRE 经典

- **[Site Reliability Engineering: How Google Runs Production Systems](https://sre.google/sre-book/table-of-contents/)** _(2016, book - Google)_ —— SRE 圣经，免费在线。SLA/SLO/SLI、Error Budget、Postmortem 等概念奠基。
- **[The Site Reliability Workbook](https://sre.google/workbook/table-of-contents/)** _(2018, book - Google)_ —— SRE 实战手册。
- **[Building Secure & Reliable Systems](https://sre.google/books/building-secure-reliable-systems/)** _(2020, book - Google)_ —— 安全 + 可靠性。

## DataOps / DRE

- **[The DataOps Manifesto](https://dataopsmanifesto.org/)** _(manifesto)_ —— DataOps 18 条原则。
